Программа позволяет составить список задач, с подзадачами любого уровня вложенности. Завершенные задачи можно скрыть. Каждую задачу можно запустить и остановить. При выводе списка задач время из подзадач учитывается в старшей задаче. 

- В файле main.cpp пропишите вашу часовую зону, локаль и желаемое имя файла БД.
- Соберите программу через cmake и запустите. В каталоге, где находится выполняемый файл, будет создан файл БД, в котором будут храниться все данные. 
- Вводите команды в консоли. Команда "помощь" выведет информацию по остальным командам.
